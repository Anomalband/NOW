# NOW Mobile (Flutter)

NOW mobil uygulamasi 24 saatlik sosyallesme akisini uctan uca sunar:
- Kayit
- Gunluk vitrin (kamera)
- Quest secimi
- Radar eslesme
- Match sohbet
- Proof gonderme
- Complete onayi
- Karma gecmisi

## Calistirma
```bash
cd mobile
flutter pub get
flutter run
```

## Test
```bash
flutter analyze
flutter test
```

## APK
```bash
flutter build apk --debug
flutter build apk --release
```

Debug APK:
- `build/app/outputs/flutter-apk/app-debug.apk`
